# Sistem Informasi Perdesaan

Projek ini merupakan sistem informasi perdesaan berbasis web yang dibuat untuk keperluan akademik.
Projek ini masih di kembangkan dan akan makin berkembang dengan seiringnya waktu.

## Fitur Utama
- Login Admin & Warga
- Laporan Warga
- Transparansi Riwayat Laporan
- Proses & Cetak Laporan
- Manajemen Data Penduduk (CRUD)

## Teknologi
- PHP Native
- MySQL
- Bootstrap 5

## Cara Instalasi
1. Clone repository ini
2. Pindahkan ke folder web server (htdocs / www)
3. Import database `database/db_desa.sql`
4. Buat file `config/koneksi.php` berdasarkan `koneksi_example.php`
5. Jalankan di browser
